# Master Spec

## Jednozdaniowa definicja produktu

System ERP to nowoczesny, wewnętrzny system ofertowo-operacyjny klasy premium B2B, który centralizuje pracę na kontach, klientach, ofertach, pozycjach, statusach, blokadach, snapshotach cen i dokumentach, z docelowym wdrożeniem jako modularny plugin WordPress.

## Filary produktu

- spójny workflow
- zrozumiały model danych
- solidne uprawnienia
- dobra obsługa dużej liczby funkcji
- nowoczesny UX
- gotowość na rozbudowę

## Filary wdrożenia

- analiza przed kodem
- prototyp będący bazą do produkcji
- architektura modułowa
- zero zgadywania
- silna kontrola regresji
- równoległe myślenie o desktop i mobile

## Filary jakości

- bezpieczeństwo
- wydajność
- czytelność
- przewidywalność
- brak regresji
- dokumentowalność
